The protocol

איך WhatsApp Web עובד?

הטלפון שלך הוא המאסטר. כל מכשיר אחר הוא Linked Device.

📱 Your phone — master, holds the keys │ │ ◄── pairing handshake via QR │ ▼ 🔗 Linked Device #1 WhatsApp Web (Chrome) 🔗 Linked Device #2 WhatsApp Desktop 🔗 Linked Device #3 your Hub (Baileys) ← we're building this

כל Linked Device מקבל הצפנת E2E נפרדת. הטלפון מאשר אותו פעם אחת — מכאן הוא עצמאי.

code example

שליחה מ-Base44

// base44/functions/send-whatsapp/index.ts
const HUB_URL   = Deno.env.get("WA_HUB_URL")!;
const HUB_TOKEN = Deno.env.get("WA_HUB_TOKEN")!;

Deno.serve(async (req) => {
  const { to, text } = await req.json();

  const r = await fetch(`${HUB_URL}/api/messages/send/text`, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${HUB_TOKEN}`,
      "Content-Type":  "application/json",
    },
    body: JSON.stringify({ to, text }),
  });

  return new Response(await r.text(), { status: r.status });
});

code example

Webhook מאומת חתימה

import { createHmac, timingSafeEqual } from "node:crypto";

const SECRET = Deno.env.get("WA_HUB_SECRET")!;

Deno.serve(async (req) => {
  const body = await req.text();
  const got  = req.headers.get("x-hub-signature") ?? "";
  const want = "sha256=" + createHmac("sha256", SECRET)
                            .update(body).digest("hex");

  // ⚠️ חובה — constant-time compare
  if (got.length !== want.length ||
      !timingSafeEqual(Buffer.from(got), Buffer.from(want))) {
    return new Response("invalid signature", { status: 401 });
  }

  const event = JSON.parse(body);
  // ... your logic ...
  return new Response("ok");
});

אבטחה לפרודקשן

  • HUB_TOKEN · 256 ביט אקראיים · Bearer בלבד
  • constant-time compare · מונע timing attacks
  • WEBHOOK_SECRET · HMAC-SHA256 על כל webhook
  • Rate limit · 120 req/min ברירת מחדל
  • Loopback בלבד · :3060 לא חשוף — רק דרך Tunnel
  • systemd hardening · NoNewPrivileges + ProtectSystem
  • unattended-upgrades · תיקוני אבטחה אוטומטיים
  • ניטור · Uptime Robot על /healthz
סיבוב Token: סקריפט מוכן ב-scripts/rotate-token.sh · 30 שניות downtime.

heads up

גיצ'ות לדעת מראש

Baileys 7 + WhatsApp LID

החל מ-2025, WhatsApp מציגים זהויות @lid במקום מספרי טלפון, כדי למנוע harvesting. ה-Hub מנסה לחלץ phone דרך senderPn / remoteJidAlt ונופל ל-LID אם אין.

Quick Tunnel URL מתחדש

בכל restart של cloudflared — URL חדש. מצוין לדמו, לפרודקשן השתמש ב-Named Tunnel עם דומיין קבוע.

Linked Device timeout

אם הטלפון לא היה online 14+ יום, ה-session פג. צריך לסרוק QR מחדש.

חובה unattended-upgrades

בלי תיקוני אבטחה אוטומטיים, אחרי 3 חודשים השרת שלך הוא יעד קל. apt-get install -y unattended-upgrades.
